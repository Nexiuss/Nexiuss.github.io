KMP-DevToy

KMP-DevToy

项目简介

这是一个基于 Kotlin Multiplatform (KMP) 技术实现的开发工具，参考了 devToy 工具的设计理念。项目采用现代化的 Compose 跨平台框架，支持在多个桌面平台上运行。

核心特性

  • ‌跨平台支持‌: 基于 Kotlin Multiplatform，实现代码在多个平台间的共享
  • ‌现代化 UI‌: 使用 Jetpack Compose 构建响应式用户界面
  • ‌桌面应用‌: 专为桌面环境优化的原生体验
  • ‌简洁高效‌: 轻量级工具，快速启动和运行

环境要求

  • Gradle 8.0 或更高版本
  • Kotlin 1.9.0 或更高版本

项目运行命令:

  • on macOS/Linux 
    ./gradlew :composeApp:run
  • on Windows 
    .\gradlew.bat :composeApp:run
